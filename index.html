Welcome to the Fellowship Pipeline!

This project is intended as a visualization tool for talent management. Originally created as a proof of concept for a hiring program, this output has since been generalized for open availability.

The pipeline creates a backdrop for organizations to discuss personnel quantities, annual hiring goals, and issues in retention. Employees can be tracked from their hiring into an intern population (part time worker or trial employee) into a junior role before joining the regular workforce.

This project is in semi-permanent beta.
